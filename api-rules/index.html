API Usage Rules

You may use the Pokémon Champions Battle Data API for personal, community, nonprofit, and commercial projects.

Attribution Required

If you display, publish, or use data from this API in a public project, you must give clear attribution.

A simple attribution is enough:

“Battle data provided by Pokémon Champions Battle Data”

Please include a link to our website or API documentation when possible.

Commercial Use Allowed

You may use this API in commercial projects, including paid apps, websites, tools, dashboards, videos, articles, and services.

Commercial use is allowed as long as you follow these rules.

No Resale or Redistribution as a Data Service

You may not resell, sublicense, or redistribute this API or its data as a standalone product.

You may not create a competing public API, database dump, bulk download service, or paid data feed using our data without written permission.

You may not sell access to our raw API responses or present our API data as your own original data source.

Caching Is Allowed

Reasonable caching is allowed to improve performance and reduce repeated requests.

Cached data should not be used to create a permanent mirror, archive, or replacement for this API.
