Pokémon index

The index is the fastest starting point. It contains searchable Pokémon metadata, sprite paths, type data, available formats, and precomputed battle summaries.

GET /data/pokemon-index.json
pokemonArray of Pokémon records.
nameDisplay name used by the Explorer.
typesPrimary type list, for example Dragon and Ground.
formsMetadata rows with form, types, combined abilities, base stats, and total stats.
summariesByFormatSmall battle-data summaries for Singles and Doubles when available.
battleDataCsvsCSV file paths for complete ranked battle data tables.

Battle data CSV

Fetch one Pokémon and one format to read the complete ranked battle table.

GET /pokemon_champions_assets/battle_data/Doubles/garchomp.csv
GET /pokemon_champions_assets/battle_data/Singles/garchomp.csv

Rows are grouped by category: moves, held items, teammates, stat alignments, stat spreads, and abilities.

pokemon,category,rank,name,percentage,stat_up,stat_down,hp_points,attack_points,defense_points,sp_atk_points,sp_def_points,speed_points,source_time_seconds

Metadata CSV

Fetch form name, typing, combined abilities, base stats, total stats, and sprite reference for a Pokémon.

GET /pokemon_champions_assets/metadata/Garchomp.csv
title,base_name,saved_name,types,abilities,image_path,form,hp,atk,def,spa,spd,spe,total

Sprites and type icons

Images can be used directly in websites, dashboards, Discord bots, and analysis tools.

GET /pokemon_champions_assets/pokemon/Garchomp.png
GET /pokemon_champions_assets/types/Dragon.png

JavaScript example

const index = await fetch('/data/pokemon-index.json').then(response => response.json());
const pokemon = index.pokemon.find(entry => entry.name === 'Garchomp');
const csv = pokemon.battleDataCsvs.find(file => file.format === 'Doubles');
const battleData = await fetch(csv.path).then(response => response.text());

Common use cases

  • Show the most common moves and items for a Pokémon.
  • Find Pokémon by type, ability, or stat profile.
  • Compare Singles and Doubles usage data.
  • Power team builders, matchup pages, or metagame dashboards.
