Advanced Search

Chain filters with spaces or commas. For example: Fake Out, spe>=100 shows all Pokemon that can learn Fake Out and have a Speed stat of 100 or higher.

Quick search
Simply type a name, type, ability, move, item, or nature.
Target Search
To search only within a certain category, type the category before the keyword. For example, type=Dragon searches the data for Dragon-type Pokemon; otherwise, it would also show Pokemon with moves such as Dragon Dance.

Rank Filter
